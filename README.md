# FGDCookingRhythmGame

Using Unity Version 2022.3.43f1

## Contributions
- If you start work on an issue, make a new branch for that issue and work in a separate folder and in a test scene.
- When the work is completed, make a pull request, and once it is approved it will be merged in.
- Before your pull request, make sure that you are using the latest version of main on your branch.
